# Budget Tracker

A simple budget tracking web app built with React and Vite.

Users can:
- Create a monthly budget
- Add multiple expenses
- See their balance update in real time
- Remove expenses
- Instantly know when they are overspending

## Tech Stack
- React
- Vite
- JavaScript
- Tailwind CSS

## What I Learned
- Managing state across multiple components
- Conditional rendering based on user flow
- Working with arrays in React state
- Building UI based on derived data
